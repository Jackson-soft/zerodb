# zerodb-driver

## import
```
<dependency>
  <groupId>com.dfire.zerodb</groupId>
  <artifactId>zerodb-driver</artifactId>
  <version>1.0.0-SNAPSHOT</version>
</dependency>
```

## config
```
driverClassName=com.dfire.zerodb.driver.Driver
url=jdbc:zerodb://zerodb.app.2dfire-daily.com/dbName?clusterName=daily(日常)
#url=jdbc:zerodb://keeper-middleware.app.2dfire-daily.com/dbName?clusterName=public(线上)
username=user
password=user
initialSize=100

as usual
```
